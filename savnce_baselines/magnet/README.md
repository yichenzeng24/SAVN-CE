# Memory-Augmented Goal descriptor Network (MAGNet)

## Details 
This folder provides the code of the model used in the 
[Semantic Audio-Visual Navigation in Continuous Environments](https://arxiv.org/pdf/2012.11583.pdf) paper.
For training and evaluation commands, see the scripts in the repository root: `pretraining.sh`, `eval.sh`, `train.sh`, and `test.sh`.
