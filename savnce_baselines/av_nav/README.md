# Audio-Visual Navigation (AV-Nav) Model

## Details

This folder contains the `AV-Nav` baseline, an audio-visual navigation model adapted from [SoundSpaces: Audio-Visual Navigation in 3D Environments](https://arxiv.org/pdf/1912.11474.pdf). For training and evaluation commands, see the scripts in the repository root: `pretraining.sh`, `eval.sh`, `train.sh`, and `test.sh`.
